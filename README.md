Machine learning analysis examining maternal and socioeconomic risk factors associated with low birth weight using logistic regression, decision trees, and KNN models.
# Project Title
Low Birth Weight Risk Analysis Using Machine Learning

# Overview
Low birth weight is a major public health concern associated with higher risks of infant mortality, developmental challenges, and long-term health complications.
This project applies supervised machine learning models to examine maternal, behavioral, and socioeconomic factors associated with low birth weight.
The analysis evaluates how different machine learning models perform in predicting the likelihood of low birth weight outcomes.

# Objectives

- Identify key risk factors associated with low birth weight

- Build predictive models to classify birth outcomes

- Compare performance across different machine learning algorithms

 -Models Used

# The following supervised learning models were implemented:

Logistic Regression

Logistic Regression with threshold adjustment

Decision Tree

Tuned Decision Tree

K-Nearest Neighbor (KNN)

# Model performance was evaluated using:

Accuracy

Precision

Recall

F1 Score
